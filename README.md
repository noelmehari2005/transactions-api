# Transactions-API


What were the new things you learned in this activity?
I learned how to create a cluster in mondoDB, create an app in heroku, and learned the difference between a container terminal and a system terminal. 

What is the purpose of the seed.js program?
Its purpose is to show us how data is transfered from the server to the database. (It creates a mock database to do that as an example)

What was the most dificult thing to do in this activity?
Definelty working with mondoDB and Heroku for the first time.

How would you say you were prudent in this assignment?
Just following the instructions that are on the repo and making sure I did EXACTLY as I was instructed.

How would you say you need to be prudent when developing this kind of web applications?
I think especially with transfering data it becomes really sensitve and prone to error so making sure that their are failsafes to these would be a wise course of action.
